## 💁🏻‍♂️ Service introduce

> 동네마트 배달 앱

**코코마트 - HappyBuyer**는 부모님께서 운영하시는 마트에서 모바일을 통한 배달 서비스를 제공하기 위해 만들어진 앱 서비스입니다. 

실제 마트를 이용하는 주 연령층을 고려하여 직관적 사용자 인터페이스와 간단한 주문 과정을 통해 누구나 이용하기 쉬운 앱을 만들려고 노력했습니다.

🗓 개발 기간: 2021.12 ~ 2022.05
